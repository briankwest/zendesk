Zendesk AI Agent Functions

This document provides an overview of the functions available for the Zendesk AI Agent integration.

Available Functions:

  • Create a new support ticket
    Function create_ticket
    Parameters subject (required), comment_body (required), requester_name, requester_email, priority, tags, custom_fields
    Description Creates a new support ticket in Zendesk.
  • Update an existing support ticket
    Function update_ticket
    Parameters ticket_id (required), status, priority, comment_body, public, tags, custom_fields
    Description Updates an existing support ticket in Zendesk.
  • Close a support ticket
    Function close_ticket
    Parameters ticket_id (required), comment_body, public
    Description Closes a support ticket in Zendesk by setting its status to 'closed'.
  • Add a comment to a ticket
    Function add_comment
    Parameters ticket_id (required), comment_body (required), public
    Description Adds a comment to an existing support ticket in Zendesk.
  • Retrieve ticket information
    Function get_ticket
    Parameters ticket_id (required)
    Description Retrieves information about a specific ticket in Zendesk.
  • Verify support PIN
    Function verify_support_pin
    Parameters caller_phone_number (required), entered_pin (required)
    Description Verifies the caller's support PIN against the stored PIN in Zendesk.
  • Retrieve current user's tickets
    Function get_current_user_tickets
    Parameters caller_phone_number (required), status, priority
    Description Retrieves ticket numbers for the authenticated user based on their phone number.

For more information, visit the GitHub repository.
